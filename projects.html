Dáil Éireann Summarization

Nov 12
❮ ❯

Main Concepts:
This project will focus on creating an easier method to inform yourself, It will focus primarily on analysing and summarising debates from the Dail that are posted on the Dail.ie website. Methods:

  1. Collect information from Dail.ie by web scaping to provide text for the following step to analyse.
  2. Use prebuilt python models to analyse and get the sentiment of the statements made.
  3. Fine tune the model to provide more accurate interpretations of Dail statements.
  4. Summarize the debates using Extractive and Abstractive interpretations using python libraries.
  5. Present the results of the text analysis with a user friendly GUI.
Expected Outcome: One single platform offering an abbreviated overview of politicians views and topics they have discussed within the Dail.
